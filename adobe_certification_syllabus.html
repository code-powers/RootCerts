Adobe Photoshop & InDesign

Certification Preparation Course

Course Goal: Prepare students for Adobe Certified Professional exams in Visual Design (Photoshop) and Print & Digital Media Publication (InDesign)

Course Overview

This course provides comprehensive training in Adobe Photoshop and InDesign, directly aligned with Adobe Certified Professional exam objectives. Students will develop industry-standard skills through hands-on projects that mirror real-world design challenges and certification requirements.

Target Certification Hours: 150+ hours of instruction and hands-on experience per Adobe standards

Exam Format: 50-minute live-in-app assessments for both Photoshop and InDesign

Certification Badge System

Badge #1: Design Industry Foundations

Aligns with Exam Objective 1: Working in the Design Industry

  • Identify purpose, audience, and design requirements
  • Demonstrate knowledge of copyright and licensing
  • Apply basic design principles and typography
  • Communicate effectively about design plans
Certification Project: Client Brief Analysis - Create a design solution for a specified target audience demonstrating understanding of demographics, accessibility, and appropriate output formats.

Badge #2: Photoshop Interface & Setup Mastery

Aligns with Exam Objective 2: Project Setup and Interface

  • Navigate Photoshop workspace and customize interface
  • Create documents with appropriate settings for intended use
  • Import and manage digital assets
  • Use guides, grids, and rulers for precise layouts
Certification Project: Multi-Format Campaign Setup - Create three document versions (web, print, mobile) of the same design with appropriate resolution, color space, and dimensions.

Badge #3: Photoshop Organization & Workflow

Aligns with Exam Objective 3: Organizing Documents

  • Master layer management and organization
  • Use selection tools and modify selections
  • Apply masks and adjustment layers
  • Organize assets for efficient workflow
Certification Project: Complex Photo Manipulation - Create a composite image using minimum 10 layers, demonstrating advanced selection techniques, masking, and non-destructive editing.

Badge #4: Photoshop Visual Elements Creation

Aligns with Exam Objective 4: Creating and Modifying Visual Elements

  • Use drawing and shape tools effectively
  • Apply and modify effects and styles
  • Work with text and typography
  • Create and edit vector graphics
Certification Project: Brand Identity Package - Design logo, business card, and letterhead demonstrating vector creation, text formatting, and effect application.

Badge #5: Photoshop Publishing & Output

Aligns with Exam Objective 5: Publishing Digital Media

  • Prepare images for different output formats
  • Save and export in appropriate file formats
  • Optimize images for web and print
  • Manage color profiles and resolution
Certification Project: Cross-Platform Portfolio - Export the same design optimized for web (JPG/PNG), print (PDF), and social media platforms with appropriate compression and dimensions.

Badge #6: InDesign Interface & Document Setup

Aligns with InDesign Exam Objective 2: Project Setup and Interface

  • Navigate InDesign workspace and panels
  • Create documents with proper page setup
  • Understand facing pages, spreads, and master pages
  • Manage document presets and templates
Certification Project: Multi-Page Publication Setup - Create a 12-page magazine template with master pages, proper margins, columns, and facing page layout.

Badge #7: InDesign Organization & Layout

Aligns with InDesign Exam Objective 3: Organizing Documents

  • Manage layers and page organization
  • Create and apply master pages
  • Use guides, grids, and alignment tools
  • Organize complex multi-page documents
Certification Project: Annual Report Layout - Design a 16-page report with consistent master pages, organized layers, and professional page flow demonstrating advanced document organization.

Badge #8: InDesign Elements & Typography

Aligns with InDesign Exam Objective 4: Creating and Modifying Document Elements

  • Create and manipulate text and graphic frames
  • Apply character and paragraph formatting
  • Create and manage styles
  • Import and link graphics effectively
Certification Project: Restaurant Menu Design - Create a tri-fold menu demonstrating advanced typography, style creation, image placement, and frame manipulation.

Badge #9: InDesign Publishing & Production

Aligns with InDesign Exam Objective 5: Publishing Documents

  • Prepare documents for print and digital output
  • Export PDFs for various purposes
  • Package projects for handoff
  • Understand prepress and production requirements
Certification Project: Professional Print Production - Create a brochure ready for commercial printing, including proper bleeds, color management, and packaging for vendor submission.
Final Certification Portfolio

Students will compile a comprehensive portfolio demonstrating mastery of all certification objectives. This portfolio will include:

  • Three advanced Photoshop compositions showcasing visual design skills
  • Two complete InDesign publications demonstrating layout and typography mastery
  • Process documentation showing design thinking and technical execution
  • Self-assessment against Adobe certification criteria
Assessment Schedule

Weekly Expectations: Minimum 2 graded assignments per week including badge assessments, practical exercises, or project milestones.

Badge Timeline: Complete 1-2 badges every two weeks, with final portfolio due two weeks before certification exam date.

Practice Exams: Monthly certification practice tests using official Adobe preparation materials.

Academic Integrity

Plagiarism Policy: Any instance of cheating or plagiarism will result in an F for the assignment and immediate referral to administration for disciplinary action.

  • 1st Offense: Assignment redo at Level 1
  • 2nd Offense: F on assignment, no redo, Level 2 intervention
  • 3rd Offense: F on assignment, no redo, parent conference and administrative hearing

This includes AI-generated work on assignments.
